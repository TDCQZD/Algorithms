## 面试切题四件套
- Clarification(询问题目细节、边界条件、可能的极端错误情况)
- Possible solutions(所有可能的解法都和面试官沟通一遍)
    - Compare Time & Space Complexity (时间复杂度&空间复杂度)
    - Optimal Solution(最优解)
- Coding (写代码)
- Test cases (测试用例)

## 个人学习方法
> 首先学习Data Structure，再专项练习Algorithm，最后LeetCode刷题。
- Data Structure:
    * 理论
    * 面试题实战(切题四件套)
    * 模块化练习
- Algorithm：
    * 复杂度分析
    * 模块化可以练习
- Leetcode 刷题
- 学习资料
    * LeetCode
    * 相关书籍
